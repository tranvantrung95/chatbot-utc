# Infographic Prompt: SƠ ĐỒ CA SỬ DỤNG TỔNG QUAN

## Configuration
- **Layout:** hub-spoke
- **Style:** corporate-memphis
- **Aspect Ratio:** 16:9
- **Language:** Vietnamese (tiếng Việt)
- **Target Tool:** Midjourney / DALL-E 3 / ComfyUI

---

## Prompt (English — for image generation tools)

```
Create a hub-spoke use case diagram infographic for UTC Assistant system in Vietnamese.

LAYOUT: Hub-spoke. Two actor figures: left = "Sinh viên" (blue), right = "Quản trị viên" (orange). Center spoke connects to use case bubbles: for Student — "Tra cứu nội quy", "Đánh giá câu trả lời", "Xem lịch sử hội thoại". For Admin — "Đăng nhập", "Xem thống kê", "Cập nhật dữ liệu", "Quản lý tài liệu". Dashed arrow from "Tra cứu nội quy" to central "Pipeline RAG" bubble, and from "Cập nhật dữ liệu" also to "Pipeline RAG".

STYLE: Corporate Memphis — flat vector, bold geometric shapes, human figure icons, vibrant accent colors on white background.

TEXT (Vietnamese):
- Title: "Hình 4.1: Sơ đồ ca sử dụng tổng quan hệ thống UTC Assistant"
- Actor labels and use case names as above
- "include" relationship markers on dashed lines

ASPECT: 16:9 landscape.
```

---

## Usage Instructions

1. Copy the prompt above
2. Paste into your image generation tool (Midjourney, DALL-E, Stable Diffusion, ComfyUI)
3. For Midjourney: add `--ar 16:9 --style raw` at the end
4. For DALL-E: set aspect ratio to `16:9`
5. Generate 2-4 variations and pick the best one
6. Save as PNG and insert into the report at the corresponding figure position
