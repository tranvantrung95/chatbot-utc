# Infographic Prompt: PHÂN LOẠI BIẾN THỂ RAG

## Configuration
- **Layout:** tree-branching
- **Style:** corporate-memphis
- **Aspect Ratio:** 16:9
- **Language:** Vietnamese (tiếng Việt)
- **Target Tool:** Midjourney / DALL-E 3 / ComfyUI

---

## Prompt (English — for image generation tools)

```
Create a tree-branching infographic showing RAG variants classification in Vietnamese.

LAYOUT: Tree branching. Root node: "RAG". Three main branches: "Naive RAG" (basic, blue), "Advanced RAG" (enhanced, green), "Modular RAG" (flexible, orange). Under each: bullet points listing features. Naive: Indexing → Embedding → Retrieval → Generate. Advanced: +Query Expansion, +Hybrid Search, +LLM Reranker, +MMR. Modular: +Memory Module, +Routing Module, +Fusion Module.

STYLE: Corporate Memphis — flat vector, bold colors, rounded shapes, playful but professional. Tree trunk and branches as smooth curving lines.

TEXT (Vietnamese):
- Title: "Hình 3.2: Phân loại các biến thể RAG"
- Branch labels and feature lists as above

ASPECT: 16:9 landscape.
```

---

## Usage Instructions

1. Copy the prompt above
2. Paste into your image generation tool (Midjourney, DALL-E, Stable Diffusion, ComfyUI)
3. For Midjourney: add `--ar 16:9 --style raw` at the end
4. For DALL-E: set aspect ratio to `16:9`
5. Generate 2-4 variations and pick the best one
6. Save as PNG and insert into the report at the corresponding figure position
