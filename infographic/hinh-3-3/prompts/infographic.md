# Infographic Prompt: QUY TRÌNH NAIVE RAG

## Configuration
- **Layout:** circular-flow
- **Style:** ikea-manual
- **Aspect Ratio:** 16:9
- **Language:** Vietnamese (tiếng Việt)
- **Target Tool:** Midjourney / DALL-E 3 / ComfyUI

---

## Prompt (English — for image generation tools)

```
Create a circular flow infographic of the Naive RAG process in Vietnamese, IKEA manual style.

LAYOUT: Circular-flow with 4 numbered steps around a circle. Step 1 (top-right): "Indexing - PDF→Chunks→DB" with document icon. Step 2 (bottom-right): "Embedding - Text→Vector 1024d" with grid/vector icon. Step 3 (bottom-left): "Retrieving - Query→Top-K Chunks" with search icon. Step 4 (top-left): "Generating - Context+LLM→Answer" with chat bubble icon. Curved arrow from Step 4 back to Step 3 showing "feedback loop". Center: "Naive RAG Pipeline".

STYLE: IKEA manual — minimal line art, black outlines on white, single accent color (IKEA blue #0058a3), simple stick-figure icons, instructional feel. No shading, flat design.

TEXT (Vietnamese):
- Title: "Hình 3.3: Quy trình Naive RAG: Indexing → Retrieval → Generation"
- Step labels as above
- Center text: "Naive RAG Pipeline"

ASPECT: 16:9 landscape.
```

---

## Usage Instructions

1. Copy the prompt above
2. Paste into your image generation tool (Midjourney, DALL-E, Stable Diffusion, ComfyUI)
3. For Midjourney: add `--ar 16:9 --style raw` at the end
4. For DALL-E: set aspect ratio to `16:9`
5. Generate 2-4 variations and pick the best one
6. Save as PNG and insert into the report at the corresponding figure position
