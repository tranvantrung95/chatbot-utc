# Infographic Prompt: USE CASE QUẢN TRỊ VIÊN

## Configuration
- **Layout:** hub-spoke
- **Style:** corporate-memphis
- **Aspect Ratio:** 9:16
- **Language:** Vietnamese (tiếng Việt)
- **Target Tool:** Midjourney / DALL-E 3 / ComfyUI

---

## Prompt (English — for image generation tools)

```
Create a detailed use case diagram infographic for Admin interactions in Vietnamese.

LAYOUT: Hub-spoke. Center: "Quản trị viên" figure icon. Spokes to 5 use cases: "Đăng nhập (token-based)", "Xem Dashboard thống kê", "Upload tài liệu PDF mới", "Re-index cơ sở tri thức", "Test truy vấn". Below: sub-flow showing "Upload → OCR + Chunking Pipeline → ChromaDB" with "include" relationship.

STYLE: Corporate Memphis — flat illustration, admin character with gear/tech elements, use case bubbles in navy and amber.

TEXT (Vietnamese):
- Title: "Hình 4.3: Sơ đồ ca sử dụng chi tiết – Nhóm chức năng quản trị"
- All labels in Vietnamese

ASPECT: 9:16 portrait.
```

---

## Usage Instructions

1. Copy the prompt above
2. Paste into your image generation tool (Midjourney, DALL-E, Stable Diffusion, ComfyUI)
3. For Midjourney: add `--ar 9:16 --style raw` at the end
4. For DALL-E: set aspect ratio to `9:16`
5. Generate 2-4 variations and pick the best one
6. Save as PNG and insert into the report at the corresponding figure position
