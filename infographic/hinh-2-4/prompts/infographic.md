# Infographic Prompt: CHIẾN LƯỢC PROMPT ENGINEERING

## Configuration
- **Layout:** comparison-matrix
- **Style:** corporate-memphis
- **Aspect Ratio:** 16:9
- **Language:** Vietnamese (tiếng Việt)
- **Target Tool:** Midjourney / DALL-E 3 / ComfyUI

---

## Prompt (English — for image generation tools)

```
Create a corporate Memphis style comparison infographic of Prompt Engineering strategies in Vietnamese.

LAYOUT: Comparison matrix. 5 strategy cards in a row: Zero-shot, Few-shot, Chain-of-Thought, Self-Consistency, Tree-of-Thoughts. Each card has: strategy name (bold, large), 1-line description, accuracy indicator (stars 1-5), token usage indicator, best-use icon. Arrow flow from left "Người dùng (câu hỏi)" through all strategies to right "LLM → Trả lời".

STYLE: Corporate Memphis — flat vector illustration, vibrant colors (coral, teal, mustard, navy), geometric shapes, abstract human figures, clean sans-serif typography.

TEXT (Vietnamese):
- Title: "Hình 2.4: Sơ đồ minh họa các chiến lược Prompt Engineering"
- Cards: Zero-shot (1★, ít token), Few-shot (3★, vừa), CoT (4★, nhiều), Self-Consistency (5★, rất nhiều), ToT (5★, rất nhiều)

ASPECT: 16:9 landscape.
```

---

## Usage Instructions

1. Copy the prompt above
2. Paste into your image generation tool (Midjourney, DALL-E, Stable Diffusion, ComfyUI)
3. For Midjourney: add `--ar 16:9 --style raw` at the end
4. For DALL-E: set aspect ratio to `16:9`
5. Generate 2-4 variations and pick the best one
6. Save as PNG and insert into the report at the corresponding figure position
