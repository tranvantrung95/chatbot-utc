# Infographic Prompt: USE CASE SINH VIÊN

## Configuration
- **Layout:** hub-spoke
- **Style:** corporate-memphis
- **Aspect Ratio:** 9:16
- **Language:** Vietnamese (tiếng Việt)
- **Target Tool:** Midjourney / DALL-E 3 / ComfyUI

---

## Prompt (English — for image generation tools)

```
Create a detailed use case diagram infographic for Student interactions in Vietnamese.

LAYOUT: Hub-spoke. Center: "Sinh viên" figure icon. Spokes to 5 use cases: "Gửi câu hỏi tiếng Việt", "Nhận trả lời streaming SSE", "Xem nguồn tham khảo", "Đánh giá (thumbs up/down)", "Xem câu hỏi gợi ý". Below: a sub-flow showing "Query Processing → Topic Filter → Retrieval 3 tầng → LLM Generation" with "include" relationships.

STYLE: Corporate Memphis — flat illustration, student character as abstract geometric figure, use case bubbles in teal and coral.

TEXT (Vietnamese):
- Title: "Hình 4.2: Sơ đồ ca sử dụng chi tiết – Nhóm chức năng sinh viên"
- All labels in Vietnamese

ASPECT: 9:16 portrait.
```

---

## Usage Instructions

1. Copy the prompt above
2. Paste into your image generation tool (Midjourney, DALL-E, Stable Diffusion, ComfyUI)
3. For Midjourney: add `--ar 9:16 --style raw` at the end
4. For DALL-E: set aspect ratio to `9:16`
5. Generate 2-4 variations and pick the best one
6. Save as PNG and insert into the report at the corresponding figure position
